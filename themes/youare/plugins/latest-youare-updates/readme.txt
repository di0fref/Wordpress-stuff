== Description ==

Show your latest YouAre updates on your WordPress blog

**Features**:

*   Displays your latest YouAre update(s), and when it was last updated
*   Number of updates
*   Clickable hashtags
*   Translations [via wp-config][1] (English, Spanish, Catalan)
*   Caches your tweets to improve page load times
*   Section Title
*   Customize XHTML

 [1]: http://codex.wordpress.org/WordPress_in_Your_Language
	
== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page. Set your own preferences in Settings > Latest YouAre Updates Options. Usage: <code><?php if (function_exists('get_youare_latest_updates')) get_youare_latest_updates(); ?></code> If you use <a href="http://wptheme.youare.com">YouAre Theme</a> no template/code editing required.
